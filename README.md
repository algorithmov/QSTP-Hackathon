# QSTP-Hackathon

**Submission Requirements**:
* Submission must consist of one folder containing these 4 files (it must be uploaded to the QSTP Google Drive folder):
 1. Product Requirements (pdf):
    - Problem definition, solution scope, users, technical architecture, and success metrics
 2. MVP (txt with link to demo):
    - A working demonstration accessible through the link
 3. Pitch Deck (pdf):
    - Visual narrative: problem, solution, demo walkthrough, impact, and scalability
 4. Pitch Video:
    - Recorded walkthrough of your product for asynchronous review by the judging panel

**Submission Deadline: Saturday 11:59**
